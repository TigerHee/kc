# Skeleton 骨架屏

骨架屏组件用于在内容加载时显示占位符，提供更好的用户体验。

## 基础用法

```tsx
import { Skeleton } from '@kux/design';

<Skeleton.Avatar size={32} />
<Skeleton.Node width={100} height={16} />
<Skeleton.Input width={320} height={32} />
```
