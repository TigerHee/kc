# umi project

## Getting Started

Node版本依赖
```
14.21.1
```

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Instanll @insight/cli
```bash
$ yarn global add @insight/cli
```

Init @insight/cli
```bash
$ insight init
```
